# HILS検証用 部品リスト（秋月電子通商）

本リストは [ROS HILSアーキテクチャ検討](ros_hils_architecture.md) に基づき、プロトタイプ検証に必要な機材を選定したものである。

> **注意**: 価格・在庫は変動するため、発注前に秋月電子通商の各商品ページで最新情報を確認すること。

---

## 1. 設計方針の変更に伴うBOM見直し

プロトタイプ検証を通じて、当初の「全デバイスをRP2040で統一」する方針から、インターフェース種別ごとに最適な方式を選択する方針に変更した。これにより、必要なマイコン数が大幅に削減された。

| インターフェース | 旧方式 | 新方式 | BOMへの影響 |
|----------------|--------|--------|-----------|
| シリアルセンサ (GPS/IMU) | RP2040 USB-CDC | FT234X × 2 クロス接続 | Pico削減、FT234X追加 |
| 3D LiDAR | RP2040 + W5500-EVB-Pico2 | USB-LANアダプタ + 純ソフトウェア | W5500-EVB-Pico2削除 |
| USBカメラ | RP2040 × 2 (UVC) | RP2040 × 2 (UVC) **変更なし** | — |

| 項目 | 内容 |
|------|------|
| 通販コード | 118085 |
| 価格 | 920円（税込） |
| 数量 | **2個** |
| 用途 | UVCカメラエミュレーション（Pico#1: CDC-SPI sender、Pico#2: UVC bridge） |

ピンヘッダ実装済（Pico H）を選ぶことで、はんだ付け不要でブレッドボードに直接挿せる。

| 機材 | 数量 | 状態 |
|------|------|------|
| USB-LANアダプタ | 1 | 必要 |
| LANケーブル | 1 | 必要 |

RP2040 TinyUSB UVCによるMJPEGパススルー。640x480@17fps動作確認済み。

| 機材 | 数量 | 状態 |
|------|------|------|
| Raspberry Pi Pico H | 2 | 必要 |
| USBケーブル (A-microB) | 2 | 必要 |
| ブレッドボード | 1 | 必要 |
| ジャンパーワイヤ | 一式 | 必要 |

### Phase 3: シリアルセンサHILS

GPS (NMEA) / IMUエミュレーションの動作確認。
