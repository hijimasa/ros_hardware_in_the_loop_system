# HILS検証用 部品リスト（秋月電子通商）

本リストは [ROS HILSアーキテクチャ検討](ros_hils_architecture.md) に基づき、ブレッドボード上でのプロトタイプ検証に必要な機材を選定したものである。

> **注意**: 価格・在庫は変動するため、発注前に秋月電子通商の各商品ページで最新情報を確認すること。

---

## 1. マイコンボード

### 1.1 Raspberry Pi Pico H（RP2040 / ピンヘッダ実装済）

| 項目 | 内容 |
|------|------|
| 通販コード | 118085 |
| 価格 | 920円（税込） |
| 数量 | **5個** |
| 用途 | シリアルセンサ(CDC)、I2C/SPIスレーブ(PIO)、カメラ(UVC)、エンコーダ/PWM |

ピンヘッダ実装済（Pico H）を選ぶことで、はんだ付け不要でブレッドボードに直接挿せる。

### 1.2 W5500-EVB-Pico2（RP2350 + W5500 Ethernet統合ボード）

| 項目 | 内容 |
|------|------|
| 通販コード | 130219 |
| 価格 | 2,420円（税込） |
| 数量 | **1個** |
| 用途 | 3D LiDARエミュレーション（Velodyne/Ouster互換UDPパケット送信） |

RP2350 + W5500がPicoフォームファクタに統合されたボード。別途W5500モジュールを配線する必要がなく、検証が容易。Ethernet RJ45コネクタ搭載。

### 1.3 ESP32-S3-DevKitC-1-N8R8

| 項目 | 内容 |
|------|------|
| 通販コード | 131148 |
| 価格 | 2,380円（税込） |
| 数量 | **1個** |
| 用途 | CAN通信（TWAI内蔵）によるモータドライバエミュレーション |

ESP32-S3はCAN互換のTWAIペリフェラルを内蔵しており、外付けCANコントローラ不要。USB Type-C接続。

## 2. CAN通信関連

### 2.1 CANコントローラ MCP2515-I/P（DIP18）

| 項目 | 内容 |
|------|------|
| 通販コード | 112030 |
| 価格 | 330円（税込） |
| 数量 | **2個** |
| 用途 | RP2040からSPI経由でCAN通信を行う場合の代替構成 |

ESP32-S3のTWAI内蔵で足りる場合は不要だが、RP2040でもCAN通信を検証したい場合に使用。DIPパッケージでブレッドボード対応。

### 2.2 CANトランシーバ MCP2561-E/P（DIP8）

| 項目 | 内容 |
|------|------|
| 通販コード | 112483 |
| 価格 | 160円（税込） |
| 数量 | **2個** |
| 用途 | CAN物理層（差動2線）のドライバ/レシーバ |

MCP2551後継のDIP8パッケージ。ブレッドボードに直接挿せる。ESP32-S3 TWAI/MCP2515いずれの構成でも、物理CANバスに接続するにはトランシーバが必要。

| 項目 | 内容 |
|------|------|
| 通販コード | 108677 |
| 価格 | 400円（税込） |
| 数量 | **2個** |

## 7. 検証フェーズと必要機材の対応

最小限の構成から段階的に検証を進めることを推奨する。

### Phase 1: シリアルセンサHILS（最小構成）

GPS (NMEA) エミュレーションの動作確認。最も単純で成功しやすい。

| 機材 | 数量 |
|------|------|
| Raspberry Pi Pico H | 1 |
| USBケーブル (A-microB) | 1 |
| ブレッドボード | 1 |

### Phase 2: LiDARエミュレーション

Velodyne VLP-16互換UDPパケット送信の検証。

| 機材 | 数量 |
|------|------|
| W5500-EVB-Pico2 | 1 |
| USBケーブル (A-microB) | 1 |
| LANケーブル | 1 |

### Phase 3: カメラ (UVC) エミュレーション

RP2040 TinyUSB UVCによるMJPEGパススルーの検証。

| 機材 | 数量 |
|------|------|
| Raspberry Pi Pico H | 1（Phase 1と兼用可） |
| USBケーブル (A-microB) | 1 |

### Phase 4: I2C/SPIセンサ + CAN + アナログ出力

I2Cスレーブ(PIO)、CAN通信、DAC出力の検証。

| 機材 | 数量 |
|------|------|
| Raspberry Pi Pico H | 2（I2Cスレーブ + エンコーダ/PWM） |
| ESP32-S3-DevKitC | 1 |
| MCP2561-E/P | 2 |
| MCP2515-I/P | 1〜2（RP2040 CAN構成の検証時） |
| MCP4725 DACモジュール | 1〜2 |
| ブレッドボード | 2〜3 |
| ジャンパーワイヤ | 一式 |
